# Seed Internet Emulator - Advanced Practice Project

## Project Overview
This project involves deploying and exploring the Seed Internet Emulator to understand how Internet infrastructure can be simulated using virtualisation and programmable network configuration.

## Team
- **Student 1** — Infrastructure & VM Setup
- **Student 2** — Source Code & Network Configuration

## Project Goals
- Set up a virtual lab environment using UTM and Kali Linux
- Deploy the Seed Internet Emulator
- Explore and understand the source code (network.py)
- Create a custom smaller network topology named "Advanced Practice"

## Tools Used
- **UTM** — Virtual machine software for Apple Silicon (M4)
- **Kali Linux ARM64** — Guest operating system
- **Docker** — Container runtime for the emulator
- **Python 3** — For writing topology scripts
- **Seed Internet Emulator** — Network emulation framework
